Para rodar no seu computador, você deve abrir o código "dashboards.py" na pasta "dashboard" e executar, no terminal do VScode - "streamlit run dashboards.py". Depois, você abre o código "login.php" na pasta principal e, no terminal, execute o comando: "php -S localhost:8000". Após isso, é só você abrir o código "index.html" na pasta princilal, no seu navegador e efetuar o login com os usuários e senha (podem ser alterados no código php na 6º e 7º linha).
